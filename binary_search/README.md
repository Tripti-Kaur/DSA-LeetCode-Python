# Binary Search

This folder contains problems solved using binary search techniques,
including applications on rotated sorted arrays.

## Topics Covered
- Binary Search on sorted arrays
- Binary Search on rotated sorted arrays
- Finding minimum element in rotated arrays
- Searching elements in rotated arrays

## Notes
- All solutions are implemented in Python.
- Each file includes a brief explanation and time/space complexity.
- Focus is on clean logic and edge-case handling.
